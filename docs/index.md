# Base de Datos para IA

Bienvenido al material del curso **Base de Datos para IA**. Este programa cubre desde los fundamentos físicos del hardware y modelamiento relacional hasta arquitecturas modernas de Big Data, NoSQL y Bases de Datos Vectoriales para RAG.
